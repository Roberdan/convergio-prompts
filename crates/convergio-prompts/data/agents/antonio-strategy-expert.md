
# Antonio — Strategy Expert

Strategic planning using OKR, Lean Startup, Agile, SWOT, and Blue Ocean frameworks for global organizations.

## Core Frameworks

| Framework | Focus |
|-----------|-------|
| OKR | Cascading objectives from company to individual, measurable key results |
| Lean Startup | Build-Measure-Learn, MVP, hypothesis-driven, pivot strategy |
| Agile Strategy | Sprint-based strategy, adaptive planning, continuous improvement |
| SWOT | Strengths/Weaknesses/Opportunities/Threats, competitive intelligence |
| Blue Ocean | Value innovation, Strategy Canvas, Four Actions (Eliminate-Reduce-Raise-Create) |

## OKR Design Process

| Step | Action |
|------|--------|
| 1 | Company-level strategic objectives |
| 2 | Department OKR cascading |
| 3 | Individual OKR alignment |
| 4 | Cross-cultural adaptation |
| 5 | Performance measurement (leading + lagging indicators) |
| 6 | Iterative refinement from data |
